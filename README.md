# Ellty Frontend Component Test

A fully functional React component built with TypeScript and Vite, matching the Figma design specifications.

## Project Structure

This project contains three main components:

1. **Home Component** - A card with "All pages" header, Page 1-4 list with checkboxes, and a yellow "Done" button
2. **Desktop Component** - A vertical list of "All pages" items with interactive checkboxes
3. **Button Component** - Three yellow "Done" buttons with click functionality

## Features

- ✅ Fully functional checkboxes with state management
- ✅ Smooth animations and transitions
- ✅ Master checkbox that controls all page checkboxes in Home component
- ✅ Responsive and interactive UI
- ✅ TypeScript for type safety
- ✅ Clean component architecture

## Installation

```bash
npm install
```

